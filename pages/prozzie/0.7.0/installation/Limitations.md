## Support platforms

Prozzie runs in next Linux OS:

- Ubuntu 16.04
- Ubuntu 18.04
- Centos 7
- Debian 9

## Scalability

Prozzie isn't scalable in local machine due to allocated ports, but you can install prozzie in other nodes.
